# Monster Siren In One

```
usage: main.py [-h] [-f FOLDER] [-n NAME_FORMAT] [-s] [-c CONCURRENT_TASKS]

Download songs from Monster Siren.

options:
  -h, --help            show this help message and exit
  -f, --folder FOLDER   Folder to save the downloaded songs.
  -n, --name-format NAME_FORMAT
                        File name format.
  -s, --overwrite-existing
                        Overwrite existing files (otherwise skip existing).
  -c, --concurrent-tasks CONCURRENT_TASKS
                        Number of concurrent tasks.
```
